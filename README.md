## Steps to Run

This project is built by `React + TypeScript + Vite`

```bash
git clone https://github.com/nikosamofa/reffie-me-interview-round-automation-react-ts.git
cd reffie-me-interview-round-automation-react-ts
yarn
yarn dev
```
